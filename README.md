# HireX

A web-based platform for finding reliable contractors and skilled workers across cities.

---

## Overview

HireX simplifies service discovery by connecting users with verified professionals in urban environments — searchable by location, specialization, availability, and ratings.

---

## Tech Stack

| Layer         | Technology                    |
|---------------|-------------------------------|
| Backend       | PHP, MySQL                    |
| Frontend      | HTML5, CSS3, Bootstrap        |
| Scripting     | JavaScript                    |

---

## Modules

**User**
- Register, log in, and search for services
- View worker and contractor profiles
- Initiate communication and submit feedback

**Worker / Contractor**
- Manage profiles and post service details
- Respond to customer queries
- View ratings and feedback

**Admin**
- Approve service provider registrations
- Monitor system activities
- Maintain data integrity

---

## Key Features

- Centralized service discovery across multiple cities
- Verified service providers
- Rating and feedback system
- Direct user-to-worker communication
- Responsive and user-friendly interface

---

## Conclusion

HireX modernizes urban service hiring by combining verification, feedback, and communication tools into one scalable platform — reducing effort for users while expanding opportunities for skilled professionals.
